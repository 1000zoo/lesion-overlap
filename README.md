# lesion-overlap

- centerplanes
  
  중심선을 구하고, 중심선을 따른 평면들을 저장하는 코드입니다.

- overlap

  저장된 tract들의 평면들을 불러오고, lesion을 입력받아, tract 별로 얼마나 overlap 되어있는지 확인하는 코드입니다.


- overlap_detector.py

  localhost port를 열어, lesion 파일을 업로드하고 결과를 쉽게 볼 수 있는 프로그램입니다.
